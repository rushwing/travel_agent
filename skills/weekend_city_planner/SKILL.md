# Weekend City Planner

Use this skill when the user wants a one-day or two-day city itinerary.

## Workflow

1. Extract the city, duration, interests, budget sensitivity, and mobility constraints.
2. Use travel tools for factual weather and location checks before proposing outdoor-heavy plans.
3. Group activities by district so the itinerary does not bounce unrealistically across the city.
4. Present a practical schedule with morning, afternoon, and evening blocks.
5. Mention uncertainty, closed attractions, or information that still needs verification.

## Output shape

- Trip summary
- Weather note
- Day-by-day plan
- Transport note
- Cost or effort caveat
